The Problem

Hundreds of medical forms stranded in legacy software needed to be incorporated into an online solution for medical examiners, and quickly.

The Solution

Ad Hoc built a lightweight form markup language that made it easy to add forms to a flexible, responsive web application for medical examiners. This solution replaced a legacy desktop application with a web-based solution that is never out of date, and can run in any platform with a modern web browser. Medical evalutations can be entered on an smart phone as easily as on a desktop.
